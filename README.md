# 猜拳五子棋

一个把石头剪刀布和五子棋缝在一起的双人对弈小游戏。

## 怎么玩

- 棋盘是标准的 19×19 五子棋棋盘，黑先白后
- 每回合系统会随机给你一种棋子：石头、剪刀或布
- 你可以在空格落子，也可以**用克制关系吃掉对方棋子**（石头吃剪刀、剪刀吃布、布吃石头）
- 先连成五子的一方获胜；对方有五子时，你还有一次机会用猜拳破局

## 操作

| 玩家 | 移动 | 确认落子 |
|------|------|----------|
| 玩家1 | WASD | 空格 |
| 玩家2 | 方向键 | 回车 |

也支持鼠标点击直接落子。

## 公平性

游戏内置了动态平衡机制：
- **前30手**：纯随机，保证开局公平
- **中盘**：系统会悄悄根据场上形势微调棋子类型的出现概率，让双方都有得打
- **残局**：平衡力度加大，防止一边倒
- 所有调整都很轻微，不会让你觉得“被安排了”

## 技术

纯前端，单文件 HTML，开箱即用。Canvas 绘制，emoji 棋子，没依赖任何框架。

## 打开方式

直接浏览器打开 `猜拳五子棋.html` 就行。
